# Agenda da Mãe

Aplicação web de agenda pessoal, mobile-first, offline-capable.

## Funcionalidades
- Calendário mensal e visualização semanal
- Eventos com categorias, prioridade e recorrência
- Checklist de tarefas da casa e do trabalho
- Bloco de notas diário
- Painel de resumo do dia
- Dados salvos localmente no dispositivo (localStorage)

## Como usar no celular
1. Abra o link no Chrome ou Safari
2. Chrome: menu → "Adicionar à tela inicial"
3. Safari: botão compartilhar → "Adicionar à Tela de Início"
